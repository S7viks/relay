# GAIOL

**One API key for all your AI models.** GAIOL routes requests across your connected providers (OpenRouter, Gemini, HuggingFace) so you don't overpay or underuse.

- **Quick start:** [QUICKSTART.md](QUICKSTART.md)
- **Database and auth:** [DATABASE_SETUP.md](DATABASE_SETUP.md)
- **Implementation plan:** [docs/IMPLEMENTATION_PHASES.md](docs/IMPLEMENTATION_PHASES.md)

Research / reference implementation. Reproducing paper metrics requires paid API access and following the benchmark documentation (see `eval/` or project docs where applicable).
